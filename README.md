## Reproduzierbarkeit


### Voraussetzungen
- Node.js v18 oder höher
- npm

###setup

```bash
# Repository klonen
git clone https://github.com/Alisonmm222/defenseethics.git

# Ins Projektverzeichnis wechseln
cd defenseethics

### Live-Version 

Die veröffentlichte Websitew ist über GitHub Pages erreichbar: 

https://alisonmm222.github.io/defenseethics/

```


Da es sich um eine statistische Website handelt, reicht es, die Dateien lokal über einen Webserver auszuliefern
